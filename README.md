# Job-Platform-User-Funnel-Analysis
채용 플랫폼 탐색형 유저의 지원 전환 촉진을 위한 로그 데이터 분석 (EDA, Funnel Analysis)
# 🔍 지원 버튼 앞에서 멈춘 유저들: 탐색형 유저 전환 전략 분석

> **"유저는 왜 공고만 보고 지원하지 않을까?"**
> 채용 플랫폼의 로그 데이터를 분석하여, 전체 유저의 47.8%를 차지하는 '탐색형 집단'의 이탈 원인을 규명하고 지원 전환율을 높이기 위한 데이터 기반 솔루션을 제안했습니다.

## 📌 Project Overview
- **배경**: 지원 단계 진입 전 대거 이탈하는 유저 행동 패턴 분석 필요
- **목적**: 활동 지표(체류시간, 탐색 빈도 등)와 지원 전환 간의 상관관계 도출 및 유저 세그먼트별 전략 수립
- **핵심 성과**: 2x2 매트릭스 분석을 통해 '고관여 탐색형' 유저를 정의하고, 맞춤형 게이미피케이션 및 리텐션 방안 도출

## 🛠️ Tech Stack
- **Languages:** Python
- **Libraries:** Pandas, Matplotlib, Seaborn
- **Analysis Method:** Funnel Analysis, User Segmentation (Matrix Analysis), Correlation Analysis

## 💡 Key Insights
1. **탐색의 질(Quality)이 전환을 결정**: 지원 경험이 있는 유저는 미지원 유저보다 평균 체류 시간이 **2.5배** 높으며, 탐색의 깊이가 깊을수록 전환율이 급격히 상승함.
2. **활동일수 vs 탐색 빈도**: 단순히 자주 접속하는 것보다, 한 번 접속했을 때 얼마나 밀도 있게 공고를 탐색하는지가 지원 전환과 더 강한 상관성을 보임.
3. **Target Segment**: '활동일수는 적지만 탐색 빈도가 높은' 유저를 잠재적 지원 가능군으로 설정하고 이들을 위한 개인화 추천 전략 제안.
